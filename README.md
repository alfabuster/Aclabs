# Aclabs.pro Writeups

This repository contains my **writeups for machines from the Aclabs.pro platform**.

The goal of this repository is to document my learning process while practicing:

- Web exploitation
- Linux privilege escalation
- Enumeration techniques
- Realistic penetration testing scenarios

All machines are solved independently and the writeups describe the methodology used during the attack process.

---

# Structure

Each machine has its own folder containing:

- `README.md` or writeup file
- commands used during enumeration
- exploitation steps
- privilege escalation techniques

Example structure:

```
aclabs-writeups
│
├── dragon
│   └── dragon_writeup.md
│
├── machine2
│   └── writeup.md
```
